# Career

## Purpose

This folder is the long-term career operating system for Henrik Lehtikangas.

The purpose is not only to manage an active job search, but to build a permanent knowledge base of companies, recruiters, applications, interviews, contacts and career decisions.

The repository should become more valuable over time.

---

## Philosophy

This repository stores **knowledge and decisions**.

It does **not** store every discussion, idea or brainstorm.

Thinking happens in ChatGPT.

Once something becomes useful in the future, it belongs here.

---

## Folder Structure

```text
Career/

README.md
WORKFLOW.md

Pipeline.md
People.md

Companies/

Templates/
```

### Pipeline.md

Current opportunities and their status.

### People.md

Recruiters, hiring managers, former colleagues and other relevant contacts.

### Companies/

One folder per company.

Each company becomes its own long-term knowledge base.

### Templates/

Reusable master documents such as CVs, cover letters and other career assets.

---

## Long-Term Vision

This repository should continue to grow long after the current job search ends.

It is intended to become Henrik's permanent career CRM and knowledge base.
